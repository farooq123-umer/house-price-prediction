# House Price Prediction

A machine learning project that predicts house prices by comparing multiple regression models.

## What I used
- Python
- Scikit-learn (Linear Regression, Random Forest)
- XGBoost
- Pandas, NumPy, Matplotlib, Seaborn

## What I did
- Cleaned and prepared the dataset with feature engineering
- Trained and compared Linear Regression, Random Forest and XGBoost
- XGBoost gave the best result with an RMSE of around 18,500
- Plotted feature importance and correlation heatmaps

## What I learned
This was my first end-to-end ML pipeline project. I learned how to compare 
different models and why tree-based models like XGBoost often outperform 
linear models on structured data.
